# Contributing

**This project does not accept contributions.**

Not A sim Empire™ is a proprietary, solo-architected platform. The
public repository exists for preview and visibility only — not for
collaborative development.

## What this means

- Pull requests will not be reviewed or merged
- Feature requests via issues will not be acted upon
- Unsolicited code submissions grant no rights or compensation

## How to engage

- Visit [notasim.world](https://notasim.world) for the live experience
- Read the docs in `/docs` to understand the vision
- Share or discuss the project on your own channels — we welcome the buzz

## Licensing inquiries

For partnership or licensing questions, contact through
[notasim.world](https://notasim.world).
